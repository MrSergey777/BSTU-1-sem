Блочная верстка использует блочные элементы (div, section, article и т.д.) для создания структуры страницы.
Табличная верстка использует HTML-таблицы для размещения элементов на странице
Адаптивная верстка использует медиа-запросы CSS для изменения стилей в зависимости от размера экрана устройства.
